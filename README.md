# Sleigh the Code 2.0 

This project is created for the Sleigh the Code 2.0 design and coding event.

## Description
A festive winter night scene created using Python Tkinter.  
The design includes a snowman made with geometric shapes and realistic snowflakes drawn using mathematical calculations.

## Technology Used
- Python
- Tkinter
- Math module
